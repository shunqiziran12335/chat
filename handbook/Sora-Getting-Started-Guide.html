    # 3.Sora的弱点

    Sora当前存在的弱点，OpenAI也是直接指出它可能****难以准确模拟复杂场景的物理原理，并且可能无法理解因果关系。

    比如“五只灰狼幼崽在一条偏僻的碎石路上互相嬉戏、追逐”，狼的数量会变化，一些凭空出现或消失。

    该模型有会可能****混淆提示的空间细节，例如混淆左右，可能难以精确描述随着时间推移发生的事件，例：遵循特定的相机轨迹。
